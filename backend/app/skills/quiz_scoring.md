# Quiz scoring skill

A Q/A session just ended. You will receive the topic's study notes, the previous mastery
memory (with the previous 0-100 score), and the full session transcript. Produce the updated
mastery assessment.

## score (0-100)

An honest estimate of how well the student currently knows **this topic as a whole** — not
just this session's percentage. Anchors:

- 0–20: barely started; most questions missed.
- 30–50: knows fragments; significant gaps remain.
- 60–75: solid on the core material; weak on details or edge areas.
- 80–95: strong command; only minor slips.
- 100: reserved for flawless, repeated performance across the whole topic.

Blend the previous score with this session's evidence: one great session on 3 questions
should move the score meaningfully but not jump it to 90 — coverage matters. The score **can
go down** if performance regressed. Answers whose grade was revised after a dispute count at
the revised grade. An abandoned-early session with few answers is weak evidence; move the
score less.

If a "Student's standing instructions" section is present, score mastery **of the scoped
material**: if the student asked to be quizzed only on chapter 2, the score reflects their
command of chapter 2, and out-of-scope sections don't count against them.

## mastery_notes

The memory you will rely on next session. Rewrite it fresh (do not append): which parts of
the notes the student has demonstrated they know, which parts they miss or have never been
asked about, and a one-line rationale for the score. Keep it under ~200 words, second person
neutral ("Strong on X; has not yet been tested on Y").

## session_summary

Two or three sentences summarizing this session for the student's history: questions asked,
how they did, anything notable (disputes, corrections).
